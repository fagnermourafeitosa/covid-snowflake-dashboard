# COVID Snowflake Dashboard

Dashboard interativo em Streamlit para explorar dados públicos da pandemia de COVID-19, consultados diretamente no Snowflake.

Os dados são mantidos pela [Our World in Data (OWID)](https://ourworldindata.org/coronavirus), uma das principais fontes mundiais para análises da pandemia, com registros diários de praticamente todos os países desde 2020.

Fonte do dataset:

https://raw.githubusercontent.com/owid/covid-19-data/master/public/data/owid-covid-data.csv


## App publicado

https://covid-app-dashboard-5tvaymhem2jxu3wuyspep3.streamlit.app/

## Sobre o projeto

Atividade prática da disciplina de Ciência de Dados (UNICAMP), com o objetivo de integrar ingestão de dados na nuvem, visualização interativa e publicação de um dashboard acessível na web.
